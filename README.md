1) LostArk_Faceting_Bot.exe 를 더블클릭하여 실행

<img width="296" height="425" alt="image" src="https://github.com/user-attachments/assets/a4368638-3a6a-4e6c-8528-81b893969eef" />


2) 메인 창에서 본인의 게임 세로 해상도 (1080P or 1440P) 에 해당하는 옵션을 체크

3) 반투명 오버레이 창의 O 위치들을 세공 위치(감소 능력까지) 에 정확히 맞춤
<img width="999" height="653" alt="image" src="https://github.com/user-attachments/assets/9b0f113f-5cfa-42b4-af4e-40f20e60256d" />

4) 메인 창의 START버튼을 누름

5) 맨 처음 돌에서 아무거나 누르면 잠시 후에 확률 계산 후 가장 최적의 (9/7 or 7/9기준) 옵션을 녹색으로 추천해줌
<img width="636" height="391" alt="image" src="https://github.com/user-attachments/assets/3cb29480-b534-4767-8a77-208bf95d8954" />

![GIF 2025-11-22 오후 10-59-38](https://github.com/user-attachments/assets/b8f56d3a-8542-4057-abe3-775346c4d664)


6) 도달 확률이 0이면 더이상 추천하지 않음

7) (메인 창의 Auto Reset 체크 시) 돌 목록에서 새 돌을 선택하면 자동으로 인식하여 다시 0/0/0/75%에서 시작
